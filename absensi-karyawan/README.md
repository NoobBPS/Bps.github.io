# Aplikasi Sistem Informasi Absensi Karyawan

Aplikasi ini merupakan sistem informasi absensi karyawan yang dibangun dengan HTML, CSS, dan JavaScript. Aplikasi ini memiliki tampilan modern dan responsif yang memungkinkan pengguna untuk mencatat kehadiran karyawan, mengelola data karyawan, dan melihat laporan absensi.

## Fitur Utama

- Mencatat absensi karyawan (Hadir, Izin, Sakit, Alpha)
- Manajemen data karyawan (NIP, Nama, Jabatan, Departemen)
- Dashboard statistik absensi harian
- Laporan absensi berdasarkan tanggal
- Penyimpanan data menggunakan localStorage browser
- Tampilan modern dan responsif
- Navigasi dengan ikon Font Awesome
- Animasi dan efek transisi yang halus

## Teknologi yang Digunakan

- HTML5
- CSS3 (dengan CSS Variables dan Flexbox/Grid)
- JavaScript (ES6)
- Web Storage API (localStorage)
- Font Awesome (untuk ikon)

## Tampilan Modern

- Warna gradasi yang menarik
- Kartu statistik interaktif
- Tabel dengan efek hover
- Tombol aksi dengan efek transisi
- Responsif untuk perangkat mobile dan desktop
- Efek hover dan animasi pada elemen-elemen UI

## Cara Menggunakan

1. Buka file `index.html` di browser
2. Gunakan menu navigasi dengan ikon untuk berpindah antar halaman
3. Tambahkan data karyawan di halaman "Data Karyawan"
4. Lakukan absensi di halaman "Absensi"
5. Lihat statistik di halaman "Dashboard"
6. Lihat laporan absensi di halaman "Laporan"

## Struktur File

- `index.html` - File utama antarmuka pengguna
- `style.css` - File stylesheet dengan tampilan modern
- `script.js` - File JavaScript untuk logika aplikasi
- `README.md` - Dokumentasi aplikasi

## Catatan

- Data disimpan di localStorage browser, jadi akan hilang jika cache browser dihapus
- Aplikasi ini tidak memerlukan server untuk dijalankan
- Dapat diakses melalui perangkat mobile maupun desktop
- Tampilan sudah responsif untuk berbagai ukuran layar
